plantillas de diseño

Hemos hecho a mano algunas plantillas guapos para que usted utilice. Siga adelante y haga clic en "Continuar diseños" para navegar a través de ellos. Usted puede fácilmente volver a editar su página antes de publicar. Después de publicar su página, usted puede visitar el generador de página y cambiar a otro tema. El contenido de su página será preservado.

Creación de páginas manualmente

Si prefiere no utilizar el generador automático, empujar una rama llamada 'gh-pages` a su repositorio para crear una página de forma manual. Además de apoyar el contenido HTML regular, GitHub páginas de soporte Jekyll, un blog conscientes generador de sitio simple, estático. Jekyll hace que sea fácil crear encabezados y pies de página de todo el sitio sin tener que copiarlos a través de cada página. También ofrece soporte de blogs inteligente y otras características avanzadas de plantillas.
